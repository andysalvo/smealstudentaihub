# Applied AI at Penn State -- Mission Rules

## Mission

The Smeal Student AI Hub is a student-built educational resource about AI in business. Maintained by Applied AI at Penn State, affiliated with the Smeal College of Business.

## Golden Rule

Don't over-engineer. Keep solutions simple and focused. Don't add features beyond what was asked.

## Naming

- The org is "Applied AI at Penn State" (not "Penn State Applied AI" -- per Policy AD07)
- Web folders: kebab-case (`ai-by-smeal-major/`, not `AIBySmealMajor/`)

## Commits

Conventional Commits: `feat:`, `fix:`, `docs:`, `style:`, `chore:`, `ci:`, `test:`

## Build Commands

NEVER CANCEL long-running commands. Set timeout to 180+ seconds for builds.

## Content

All content follows `content/VOICE_BRIEF.md`. Field first, AI second. Describe, don't instruct. No hype.

## Penn State Branding

- Colors: Use Tailwind brand tokens only. Never hardcode hex values.
- Cannot use: Lion Shield, athletics logos, mascot, seal, building images.
- Footer disclaimer is required on every page.
