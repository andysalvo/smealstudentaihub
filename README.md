# Smeal Student AI Hub

A student-built resource for understanding AI in business, school, and professional life. Maintained by Applied AI at Penn State.

**Live site:** https://andysalvo.github.io/smealstudentaihub/

## For students

**New here?** Read [GETTING-STARTED.md](GETTING-STARTED.md). You don't need to know how to code. Open this repo in Claude Code and it will guide you through everything you can do.

**Available workflows:**
- Create a new AI News article from 2-3 sources
- Add sources to Learn pages
- Refresh module images
- Review the site visually

See [docs/workflows/](docs/workflows/) for step-by-step guides.

## For developers

**Tech stack:** Next.js 16, React 19, TypeScript, Tailwind CSS v4, Motion (framer-motion), GitHub Pages

**Quick start:**
```bash
git clone https://github.com/andysalvo/smealstudentaihub.git
cd smealstudentaihub
npm install
npm run dev
```

**Agent instructions:** [AGENTS.md](AGENTS.md)

**Design research:** [docs/corpus/workflows/](docs/corpus/workflows/)

## What's on the site

- **10 interactive Smeal major modules** with sidebar navigation, 162 concepts, key points, per-concept images
- **4 Learn pages** on AI basics, business applications, ethics, and academic/professional use
- **AI News articles** researched and written by club members
- **22 foundational sources** with full attribution

## Content standards

All content follows the [Voice Brief](content/VOICE_BRIEF.md). Field first, AI second. Describe, don't instruct. No hype.

## Contributing
