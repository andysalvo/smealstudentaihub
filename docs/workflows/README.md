# Agent Workflows

Step-by-step guides for common site updates. Each workflow is designed to be run by a student using an AI agent (like Claude Code). No coding experience needed.

| Workflow | What it does |
|----------|-------------|
| [NEW-ARTICLE.md](NEW-ARTICLE.md) | Create a new AI News article from source URLs |
| [UPDATE-LEARN-PAGE.md](UPDATE-LEARN-PAGE.md) | Add a SourceCard to a Learn page |
| [REFRESH-IMAGES.md](REFRESH-IMAGES.md) | Download fresh Pexels images for major module cards |

All content should follow the voice brief at `content/VOICE_BRIEF.md`.
